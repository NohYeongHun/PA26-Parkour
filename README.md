# Parkour
 캐릭터 파쿠르 기능을 구현하는 프로젝트입니다.
</br>
장애물 감지, 상태 머신 기반 동작 전환, IK 보정을 통해 벽타기·매달리기·구르기 등 파쿠르 동작을 구현하는 것을 목표로 합니다.
<br></br>

## 일정

| 주차 | 작업 목록 | 비고 |
| :---: | :--- | :--- |
| 1주차 | 프레임워크, 애니메이션, 맵 에디터 | x | 
| 2주차 | 맵 오브젝트 배치, 환경 탐지, 장애물 판별 | x | 
| 3주차 | 캐릭터 상태 머신 | x | 
| 4주차 | 애니메이션 IK 적용 | x | 
| 5주차 | 장애물 넘기, 구르기 | x | 
| 6주차 | 벽타기 | x | 
| 7주차 | 매달리기(난간, 모서리) | x | 
| 8주차 | 안정화, 낙하 중 파쿠르 동작(시간 여유 시) | x |
