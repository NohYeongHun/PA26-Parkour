# Parkour
 캐릭터 파쿠르 기능을 구현하는 프로젝트입니다.
</br>
장애물 감지, 상태 머신 기반 동작 전환, IK 보정을 통해 벽타기·매달리기·구르기 등 파쿠르 동작을 구현하는 것을 목표로 합니다.
<br></br>

## 주차별 구현 내용
 
### 1주차 — 프레임워크 구현
**목표**
> 프레임워크를 준비하고 애니메이션을 구동시키며 맵 일부를 배치한다
 
**구현 내용**
- 프레임워크 준비
- 애니메이션 에디터 구현 (애니메이션 목록 재생 가능)
- 맵 에디터 구현 (맵 배치 가능)
- 캐릭터 & 맵 데이터 레벨에 로드 ( 배치한 데이터를 레벨에 로드)
---
 
### 2주차 — 환경 탐지, 장애물 판정 및 볼트 구현
**목표**
> 전방 장애물을 감지하고 종류를 구분한다.
> 
> 환경 정보를 기반으로 실행 가능한 파쿠르 동작을 결정하고, 볼트 동작(진입 - 이탈)을 실행 가능하게 한다.
 
**구현 내용**
> 

---
 
### 3주차 — 벽면 파쿠르 동작,  애니메이션 동적 보정 구현
**목표**
> 벽면을 인식하고 (진입 - 등반 - 이탈)을 구현한다.
> 
> 애니메이션의 루트 모션을 목표 위치에 맞게 동적 보정한다.

 
**구현 내용**
>

---

### 4주차 — 벽면 장애물 동작 구현 및 IK 보정
**목표**
> 벽면 장애물에 매달린 상태에서의 동작을 구현한다.
> 
> IK 보정으로 애니메이션이 지형에 맞닿는 부분을 자연스럽게 연결하고, 전체 동작을 안정화하여 데모를 완성한다.
 
**구현 내용**
> 

---

>
> 이탈 시점에 다음 장애물을 재탐지하여 다음 파쿠르 액션 결정
