## 지형의 판별 기준 고민

### 고민
- 구현 시 장애물을 넘는다, 올라갈 수 있다와 같은 판단을 어떤 기준으로 하는가?

### 기존 생각
1. 캐릭터에 박스 모양의 센서를 입히고 이 센서에 감지 + 캐릭터 정면 기준 거리, 각도를 통해 탐지된 객체의 태그를 파악한다.
2. 태그를 바탕으로 행동을 결정한다.

### 다른 생각
1. 캐릭터에 박스 모양의 센서를 입히고 이 센서에 감지 + 캐릭터 정면 기준 거리, 각도를 통해 탐지된 후보 객체를 선정한다.(Broad Phase)
2. 선정된 가장 가능성이 높은 후보 객체에 RayCast를 발사합니다.
3. RayCast를 통해 기하 정보를 추출합니다.
4. 기하 정보를 데이터로 생성합니다.
5. 데이터를 바탕으로 결정을 진행합니다.

### 또 다른 생각
1. 캐릭터 전방에 Box Shape Cast 실행
2. 선정된 가장 가능성이 높은 후보 객체 검출
3. 검출된 지형의 상단 / 전방에 레이캐스트를 수직, 수평 발사
4. 정확한 모서리의 좌표 및 법선 벡터 획득
